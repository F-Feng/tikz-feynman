TikZ-Feynman

TikZ is a fantastic and very powerful drawing package and TikZ-Feynman builds upon this to provide an easy way to produce Feynman diagrams.

I must thank Jake for providing this answer on tex.se which initially inspired me to write this package.

If you have any suggestions or have found any bugs, please feel free create a new issue or pull request here on Github.

All the documentation can be found in tikz-feynman.pdf.

TikZ-Feynman is maintained by JP-Ellis.
